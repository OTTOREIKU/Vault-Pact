# Credits

## EVE Frontier Game Data

The blueprint and crafting recipe data used in this application is derived from
EVE Frontier, a game developed by CCP hf. Game content and materials are the
property of CCP hf.

The original compilation of this data was done by TehFiend as part of the
Frontier Periscope project (https://github.com/tehfiend/Frontier-Periscope).
The data has been reformatted and restructured independently for use in
Vault Pact.
